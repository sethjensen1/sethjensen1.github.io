Get the Code

The Liquid Galaxy combines several apps for viewing content as well as libraries to support those apps in a properly maintained system. The Liquid Galaxy can run on Linux, Apple, and Windows systems (note: anecdotal evidence shows Linux to be the most stable). For your given platform, please download and install all 3 files.
